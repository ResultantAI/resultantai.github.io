The Problem

20 trucks loading pumice daily. Paper tickets getting lost (~$500/week). Handwriting illegible causing billing disputes. Manual calculations (lbs → yards → tons) done by hand. Office spending 2+ hours/day on data entry.

The Solution

Paper-to-digital ticketing system:

  • Tablet-friendly form for loader operators
  • Auto-calculates weight conversions
  • PDF ticket generated instantly (matches old paper format)
  • Real-time dashboard for office visibility
  • Works offline, syncs when connection returns
  • Full status workflow (Open → Hold → Closed → Void)
  • QuickBooks export integration

The Results

Time per ticket: 5+ minutes → 30 seconds
Lost tickets: ~$500/week → Zero
Office data entry: 2+ hours/day → Eliminated
Real-time visibility: Next day (maybe) → Instant

Owner, Beaver Pumice

What We Built

  • Platform: Custom web app + tablet interface
  • Integration: QuickBooks IIF export for accounting
  • Features: Offline mode, user roles, customer management
  • Timeline: Phase 1 (2 weeks), Phase 2 (3 weeks)
  • Ownership: Client owns all code and hosting

The Problem

MCA funding business bottlenecked by manual calling. Limited to 15-20 outbound calls per day. Document collection taking 5+ days. Zero coverage outside business hours. Inconsistent qualification across different reps.

The Solution

AI-powered outbound qualification system:

  • AI voice agent for outbound calls (100-120 calls/hour)
  • Workflow orchestration with conditional routing
  • CRM integration for full journey tracking
  • 4 conditional email sequences (Qualified, Objections, No-answer, Incomplete)
  • Automated document collection with reminders
  • TCPA-compliant call recording and consent
  • Edge case handling (hung-up calls, API timeouts, duplicates)

The Results

Call volume: 15-20/day → 100-120/hour
Qualified leads: 3X increase from baseline
Manual calling time: 75% reduction
Document collection: 5 days → 1.5 days
Coverage: Business hours → 24/7
Data capture: Inconsistent → 100% standardized
