2. **dispatcher-dashboard.png**
   - Real-time dispatcher dashboard view

3. **driver-mobile-app.png**
   - Mobile app interface for drivers
   - Should show: route list, load details, navigation options

   - Recommended size: 1200x675px

4. **customer-portal.png**
   - Customer self-service portal
   - Should show: delivery history, load requests, ticket downloads
   - Aspect ratio: 16:9
   - Recommended size: 1200x675px
